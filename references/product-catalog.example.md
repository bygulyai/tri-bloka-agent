# Product Catalog

**Сайт компании - https://example.com/#products**

Замените этот файл на каталог ваших продуктов/услуг. Формат:

---

::**ПРОДУКТЫ:

1) Название продукта 1 - https://example.com/product-1 ;
Описание продукта 1, ключевые выгоды, длительность, формат.

2) Название продукта 2 - https://example.com/product-2 ;
Описание продукта 2, ключевые выгоды, длительность, формат.

3) Название продукта 3 - https://example.com/product-3 ;
Описание продукта 3, ключевые выгоды, длительность, формат.

________________________________________________________

## Как заполнить

Для каждого продукта укажите:
- Полное название
- URL страницы продукта
- Краткое описание (2-3 предложения)
- Ключевые выгоды
- Длительность и формат
- Ценовые тарифы (если известны)

## Источник цен

Цены всегда проверяйте на живой странице продукта. Каталог - только для маршрутизации и названий, не для актуальных цен.
