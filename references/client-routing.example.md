# Client Routing

Используйте эту карту для быстрой маршрутизации клиента к продукту. Всегда проверяйте точное название и ссылку в `product-catalog.md` перед финальным ответом.

Если в заметках с консультации уже указан продукт, предпочитайте его, если он не противоречит цели клиента.

## Как заполнить

Создайте группы по категориям потребностей клиентов и укажите, какой продукт рекомендовать для каждой группы.

---

## Пример структуры маршрутизации

### Категория 1: Автоматизация и процессы

- Ключевые слова: автоматизация, процессы, CRM, интеграции, боты
- Рекомендация: `Название продукта для автоматизации`

### Категория 2: Начальный уровень

- Ключевые слова: хочет понять основы, низкая уверенность
- Рекомендация: `Название вводного продукта`

### Категория 3: Монетизация и рост

- Ключевые слова: зарабатывать больше, фриланс, свой бизнес
- Рекомендация: `Название продукта для монетизации`

### Категория 4: Профессиональные роли

- Ключевые слова: конкретная профессия, роль в компании
- Рекомендация: `Название продукта для профессий`

### Категория 5: Маркетинг и контент

- Ключевые слова: трафик, контент, визуал, продвижение
- Рекомендация: `Название продукта для маркетинга`

---

Заполните категории и продукты под ваш бизнес.
